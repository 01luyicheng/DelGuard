## 🚀 一键安装命令

### Windows (PowerShell)
```powershell
iwr -useb https://raw.githubusercontent.com/01luyicheng/DelGuard/main/install.ps1 | iex
```

### Linux/macOS (Bash)
```bash
curl -fsSL https://raw.githubusercontent.com/01luyicheng/DelGuard/main/install.sh | bash
```
